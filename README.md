# Weather Forecast Application 🌦️

## Project Overview
This project is a **Weather Forecast Application** built using **HTML, Tailwind CSS, and Vanilla JavaScript**.  
The application allows users to check **current weather conditions** and **extended forecasts** for any city or their current location.

The main goal of this project is to understand **API integration**, **DOM manipulation**, **event handling**, and **responsive UI design** using modern frontend tools.

---

## Features
- Search weather by **city name**
- Get weather for **current location** using browser geolocation
- Display **current temperature, humidity, wind speed, and weather condition**
- **5-day extended weather forecast**
- Temperature unit toggle (**°C / °F**) for today’s temperature
- **Recently searched cities dropdown** (stored using localStorage)
- Dynamic weather icons and background change based on weather condition
- Custom error messages for:
  - Invalid city name
  - Empty search input
  - API or network errors
- Fully **responsive design** (Desktop, iPad Mini, iPhone SE)

---

## Technologies Used
- **HTML5** – Structure of the application
- **Tailwind CSS (CDN)** – Styling and responsive layout
- **JavaScript (ES6)** – Application logic and API handling
- **OpenWeatherMap API** – Weather data provider
- **Local Storage** – Store recently searched cities
- **Git & GitHub** – Version control
